Google Drive Sharing URL (Tutorial: How to get Google Drive sharing URL)

By default, the audio/video file size needs to be under 100MB, otherwise Google Drive will complain it can't scan the virus and will refuse to directly output the file. If your video file size is larger than 100MB, open the generated Google Drive direct link with Google Chrome, right click the button "Download anyway", then click Copy Link Address from the context menu.

How to Get Google Drive Sharing URL

1. Right click on the media file in Google Drive and select the menu item Share

2. Click the Get Sharable link

3. Select More from the drop down list

4. Select On - Public on the web

5. Click the button Copy Link, then click Done
